## SWriteMap

SWriteMap75.rar

До недавнего времени для заливки map требовалась програма Siemens EEPROM tool. Сейчас мапы можно устанавливать гораздо проще, с помощью SWriteMap, которая сделает это в автоматическом режиме.

Перед установкой мапа открываем скачанный файл с мапами (обычно это архив zip) и извлекаем из него два файла: общий - StdFile (например, для СХ70 он имеет название CX70_1_Standard_50_0001.map), и региональный - DeltaFile (например, для СХ70 и региона Россия, он имеет название CX70_2_ru-Russian-Retail_50_0411.map). Остальные не понадобятся.

Запускаем программу, указываем пути к файлам мапов - StdFile (например, для СХ70 имеет название CX70_1_Standard_50_0001.map), DeltaFile (например, для СХ70 имеет название CX70_2_ru-Russian-Retail_50_0411.map). Подключаем ВЫКЛЮЧЕННЫЙ телефон кабелем к компу. Жмём кнопку Go в программе, и тут же коротко, но чётко тыкаем красную кнопку телефона (если всё сделано правильно, на экране телефона должна появиться надпись Service Mode). Всё остальное программа сделает сама.

Помимо такого несомненного достоинства данного метода, как простота и удобство, стоит отметить ещё отсутствие проблем с дефрагментацией EEPROM и автоматическое обновление информации о версии установленного мап. В качестве недостатка отмечу, что в моём случае программа, при использовании её в системе WindowsXP SP2 категорически отказалась работать, а из под WindowsXP SP1 отлично заработала.
